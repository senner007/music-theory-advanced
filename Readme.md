### Supported platforms:
- Windows

### Install
- install node.js
- in folder run : npm install
- npm install -g ts-node

### Compile
- npm run build

### Develop
- npm run dev

### Test
- npm test
- debug in vscode, select "Debug Current Test File" launch mode under debug

### Compile and run
- npm run prod
